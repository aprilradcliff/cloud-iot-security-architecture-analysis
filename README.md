# Cloud and IoT Security Architecture Analysis

## Overview

This repository contains cloud and IoT security analyses demonstrating how modern enterprise systems are designed, deployed, and secured.

The work focuses on cloud infrastructure deployment, IoT device security, network segmentation, and access control strategies used to protect sensitive data and systems in distributed environments.

---

## Key Areas Covered

• Cloud infrastructure deployment and security (AWS EC2)  
• IoT device security and network segmentation  
• Access control and authentication strategies  
• Risk mitigation for connected systems  
• Enterprise security architecture design
